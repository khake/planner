## Tech Stack
- **Next.js 15** (App Router)
- **React 19**
- **TypeScript**
- **Tailwind CSS** + **tailwindcss-animate**

## Database Schema (Supabase / Postgres)
**ตารางหลัก:**
- **users** – id, name, avatar_url
- **projects** – id, name, description
- **sprints** – id, project_id, name, status (planned | active | completed), start_date, end_date, goal, actual_end_date
- **tasks** – id, project_id, sprint_id (nullable = backlog), title, description, status (backlog | todo | in_progress | review | done), priority (low | medium | high | urgent), assignee_id
- **attachments** – id, task_id, file_url, file_name, file_type
- **task_comments** – id, task_id, author_name, author_id, content
- **profiles** – id (FK auth.users), name, avatar_url (sync จาก auth.users)
- **activity_logs** – id, user_id, action, target_type, target_id, details (jsonb)
**Storage:** bucket `task-artifacts` สำหรับอัปโหลดไฟล์แนบ task
**Trigger:** เมื่อมี user ใหม่ใน auth.users ให้สร้าง profile และ sync ไป users
---
## Auth & Route Protection
- **Auth:** Supabase Auth (signInWithPassword, signUp, signOut)
- **Login:** อีเมล + รหัสผ่าน
- **Register:** อีเมล + รหัสผ่าน (ต้องมี email confirmation ถ้าเปิด)
- **Middleware:** ป้องกัน /projects, /projects/*, /profile — ถ้าไม่มี session redirect ไป /login?from=...
- **หน้าแรก (/):** ถ้า login แล้ว redirect ไป /projects
- **ทุกหน้า projects:** เช็ก Supabase auth ก่อน render (redirect ถ้าไม่มี user)

## ฟีเจอร์หลัก
1. **Squads (Projects):** สร้าง/ดูรายการ, ดูรายละเอียด, ลิงก์ไป Backlog / Board
2. **Sprints:** สร้างใน Backlog, Start Sprint, Complete Sprint (ย้ายงานไม่เสร็จไป Backlog หรือ Next Sprint)
3. **Backlog:** ลาก task ระหว่าง Backlog กับ Sprint columns (drag & drop)
4. **Kanban Board:** 4 lanes (Todo, In Progress, Review, Done), ลาก task ระหว่าง lanes
5. **Task:** สร้าง/แก้ไข/ลบ, กำหนด assignee, priority, attachments (รูป/PDF), comments
6. **Profile:** แก้ไขชื่อ, เปลี่ยนรหัสผ่าน
7. **Toast:** แจ้งการกระทำสำเร็จ (เช่น สร้าง Sprint แล้ว)

## UI/UX
- **Theme:** Shopee-inspired (primary ส้ม #EE4D2D), light mode, ฟอนต์ Prompt
- **Card:** class `.card` สำหรับการ์ด (border, shadow)
- **Modal:** component ร่วมสำหรับ CreateSprint, Create Squad, Complete Sprint
- **Drag & drop:** ใช้ `dropAnimation={null}` ใน DragOverlay เพื่อไม่ให้การ์ดวิ่งกลับไปตำแหน่งเดิม
- **Header:** ปุ่มโปรไฟล์, Logout, CurrentUserTag (ชื่อผู้ใช้) ใกล้ปุ่มโปรไฟล์
- **Profile:** แบบฟอร์มมีกรอบ card, ปุ่มอยู่ใกล้ input และจัดชิดขวา
